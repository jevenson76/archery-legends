# Runbooks Index

Standard operating procedures for Archery Legends development.

## Available Runbooks

| Runbook | Purpose |
|---------|---------|
| [playtest.md](playtest.md) | Verify code changes via Studio playtest |

## When to Use

Runbooks provide step-by-step procedures for common operations. Use them when:
- Starting a new type of task
- Onboarding to the project
- Ensuring consistent verification

## Adding New Runbooks

Create a new `.md` file with:
1. Purpose section
2. Prerequisites
3. Step-by-step procedure
4. Success criteria
5. Common issues/troubleshooting
